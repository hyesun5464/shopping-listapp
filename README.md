# 🛒 쇼핑 리스트 앱

장 볼 항목을 추가하고 관리하는 간단한 웹앱입니다.

## 기능

- 항목 추가 / 삭제
- 완료 체크 (토글)
- 완료 항목 일괄 삭제
- 남은 항목 수 표시
- `localStorage`를 이용한 데이터 영구 저장

## 기술 스택

- 순수 HTML / CSS / JavaScript (의존성 없음)
- 브라우저 `localStorage`

## 실행 방법

별도의 빌드 과정 없이 `index.html` 파일을 브라우저로 열기만 하면 됩니다.

```bash
# 예시: 로컬 서버로 열기
python -m http.server 8000
# http://localhost:8000 접속
```
